# Pokémon Damage Calculator - Champions Mode (精简版)

## 📋 简介

这是 **Pokémon Damage Calculator** 的精简版本，仅包含 **Champions 模式**。

- **完整版大小**: 8.1 MB
- **精简版大小**: 3.4 MB（减少 58%）
- **支持语言**: 中文/English
- **使用方式**: 双击 `index.html` 即可使用，无需安装任何服务器

## 🚀 使用方法

### Windows
1. 双击 `index.html` 文件
2. 浏览器会自动打开

### Mac/Linux
1. 双击 `index.html` 文件
2. 或右键选择"用浏览器打开"

## ✨ 主要功能

- ✅ 完整的Champions模式计算器
- ✅ 中英文双语界面
- ✅ 收藏配置管理
- ✅ 深色主题支持
- ✅ 本地存储数据
- ✅ 所有Field区域功能完整

## 📂 包含内容

```
champions-only/
├── index.html          # 主页面（双击打开）
├── css/                # 样式文件
├── img/                # 图片资源
├── js/                 # JavaScript文件

```
